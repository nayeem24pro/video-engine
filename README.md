# 🎬 Faceless Video Engine

Automated faceless video generator using Remotion + Gemini AI.  
Generates short-form videos (30s) with AI scripts, stock footage, and TTS voiceover.  
Runs on GitHub Actions (free) and deploys to Vercel.

## ✨ Features

- 🤖 AI script generation via Google Gemini API
- 🎥 Stock videos/images from Pexels (free tier)
- 🔊 Text-to-speech using Edge TTS (free, no API key)
- 🎞️ Programmatic video rendering with Remotion
- ⏰ Scheduled generation via GitHub Actions (every 6 hours)
- 🌐 Deployable to Vercel (static hosting + serverless functions)
- 📱 Output: 1080x1920 vertical video (TikTok/Shorts/Reels ready)
